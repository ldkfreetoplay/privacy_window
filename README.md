# privacy_window
 A flutter plugin for OHOS platform, used to set up a privacy window

#example

``` dart
import 'package:privacy_window/privacy_window.dart';

  setPrivateWindow() {
    PrivacyWindow().setPrivacyWindow();
  }

  unSetPrivateWindow() {
    PrivacyWindow().unSetPrivacyWindow();
  }

```
